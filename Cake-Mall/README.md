## 使用说明

1. 将图片资源放置在对应目录中
2. 在 project.config.json 中填写您的小程序 AppID
3. 在微信开发者工具中导入项目
4. 编译预览即可

## 功能说明

- 首页：展示蛋糕列表和分类，支持按分类筛选蛋糕
- 详情页：展示蛋糕详情，可加入购物车
- 购物车：管理已选蛋糕，计算总价
